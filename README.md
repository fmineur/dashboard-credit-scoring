# Dashboard Crédit Scoring 🧠💳

Cette application Streamlit permet de visualiser l’analyse de risque de crédit pour les clients d’une banque.

## Fonctionnalités :
- Vue générale des clients et du crédit
- Facteurs d'influence SHAP (local et global)
- Comparaison avec les clients voisins
- Simulation de profil
- Création de nouveaux dossiers
